Flexbox vs Grid - Short Note

Flexbox and Grid are both CSS layout systems, but they serve different purposes.

Flexbox is one-dimensional (row or column). It is best for aligning items in a single direction, such as navigation bars, cards, or buttons.

Grid is two-dimensional (rows and columns). It is ideal for full-page layouts or sections requiring precise row/column alignment.

Which one I chose: I choose Grid to arrange a major section such as the projects or gallery layout because it provides better control over two-dimensional layouts and allows for more complex arrangements.
